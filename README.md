# SocialStudy

## Development Setup

### Backend

- Install dependencies

  ```bash
  npm i
  ```

- Rename `.env.template` to `.env` and complete the values

- Start development server
  ```
  npm run dev
  ```

### Frontend

- Install dependencies

  ```bash
  cd socialstudy/
  npm i
  ```

- Run development server

  ```
  npm start
  ```

- Go to `localhost:3000` in your browser

## Commands

- Run tests
  ```
  npm test
  ```
